🏛️ PublicLedger

**Transparent Tracking for Government Infrastructure & National Schemes**

Built for the **Pallotti National Hackfest 2026**.

## 🚀 Overview
PublicLedger is a full-stack dashboard designed to bridge the transparency gap in public fund allocation. It allows citizens to track the progress and budgets of national schemes and infrastructure projects, while providing an active accountability tool to flag financial discrepancies.

## ✨ Key Features
* **Live Dashboard & Analytics:** Tracks total budgets, expenditures, and completion rates with interactive visual matrices using Chart.js.
* **AI Audit Engine:** A Python-based algorithm that actively scans the local database for financial anomalies, automatically flagging projects where spending drastically outpaces physical completion.
* **Secure Admin Access:** Protects sensitive actions with a custom OTP-based authentication system.
* **Citizen Reporting Portal:** A secure interface for users to report data discrepancies or project delays directly to the ledger.

## 💻 Tech Stack
* **Backend:** Python, Flask
* **Database:** SQLite
* **Frontend:** HTML5, CSS3 (Glassmorphism UI), Vanilla JavaScript

## ⚙️ How to Run Locally
1. Clone the repository to your local machine.
2. Install the required dependencies: `pip install flask requests`
3. Run the backend server: `python main.py`
4. Open your web browser and navigate to `http://127.0.0.1:5000`
